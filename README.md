# Smart Link Copier

A browser extension to copy GitHub PR links, Jira tickets, or custom text links with a single keyboard shortcut (Ctrl+Shift+K).

## Installation

1. Open Chrome or Edge.
2. Go to `chrome://extensions` (or `edge://extensions`).
3. Enable Developer mode.
4. Click "Load unpacked" and select this folder.

## Usage

- Open any page (GitHub PR / Jira / any page)
- Press **Ctrl + Shift + K**
- Type custom text (or leave blank) → Enter
- The link is copied to clipboard.
